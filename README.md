
## Run Locally
Prerequisit: rust.
I was developing with rustc 1.68.0 (2c8cc3432 2023-03-06)


Clone the project

```bash
  gh repo clone bulengerkacper/lump
```

Go to the project directory

```bash
  cd lump
```

Install dependencies

```bash
  cargo build
```

Start an app

```bash
  cargo run
```
